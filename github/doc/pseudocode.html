Pseudocode Documentation

This file defines a simple pseudocode language for rendering graphics on a canvas. Below are the instructions for using the commands available in this language:

General Structure:

The file is divided into sections by commands starting with special characters:

  • The > command initializes the map and canvas properties.
  • The @ command switches to the next canvas.
  • Drawing commands follow to render shapes and lines on the canvas.

Commands:

1. Initialization and Canvas Switching:

>MAP <cell size> <n° of cells in x> <n° of cells in y> <n° layer>

Initializes a new map with the specified cell size, number of cells in x and y directions, and layer.

@TRACK

To draw the track & start line. The track color is specified in the interface (default is black), so the command C in Track is ignored.

@GRID

To draw the grid. The grid represents the cells of the map (available moves).

2. Track Drawing Commands:

W <width>

Sets the width of the track lines.

Q <xc> <yc> <endX> <endY>

Draws a quadratic curve from the current position to (endX, endY) with control point (xc, yc).

$<command>

Special track-related command such as starting line.

3. Background and Grid Commands:

F <r> <g> <b>

Sets the fill color using RGB values.

R <x> <y> <width> <height>

Draws a filled rectangle with the specified position and dimensions.

O <size>

Renders the grid outline with the specified size.

4. Checkpoints:

P <x> <y>

Draws a checkpoint at the specified coordinates.

Example:

>MAP 20 40 40 3

@TRACK
# THE BACKGROUND
F 0 0 190
R 0 0 900 900

# THE TRACK
W 90 # default width is 50
# xc yc endX endY
Q 100 200 100 500 # 4 params
Q 100 700 200 700
Q 200 700 600 700
Q 800 700 600 500
Q 500 400 600 300
Q 800 100 400 70
Q 100 30 100 150
$TART_LINE 100 300
$TART_LINE 300 700

@EXTRA


@WAYPOINTS
# to draw the waypoints by the debug mode (using max layer - 2)


@CHECKPOINTS
## CHECKPOINTS (MAIN)
F 200 0 170
#P 300 700
#P 100 200
#P 100 150
#P 200 700
#P 600 700
##P 600 500
#P 600 300
##P 400 70


@GRID
#G
O 10
